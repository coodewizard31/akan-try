## Akan Name Generator : Description
This project is a web-based application that determines a user’s "soul name" based on the Akan tradition of Ghana.
 This web application is a digital bridge to Akan cultural heritage, designed to help users discover their traditional Ghanaian "soul names" (Kradin). In the Akan naming system, every child is assigned a name based on the day of the week they were born, which is believed to influence their character and destiny. 

### Objectives
1.Identify a user's day of birth using computational logic.
2.Educate users on the specific meaning and gender-based variations of Akan names.
3.Preserve traditional naming practices through an accessible, modern interface.
